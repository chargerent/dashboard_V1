movement is set to 15 minutes. If this value is set to 0, the immediate self-check mechanism is triggered, but it should be noted that the system will still
return to the preset 15-minute automatic self-check interval after the immediate operation. Users can change the specific frequency of the movement
performing routine self-checks by adjusting this parameter. For back-end managers, if considering changing this setting value, it is recommended to keep
it at least 5 minutes to ensure a good balance between equipment operation stability and efficiency.

values can alter the interval between each self-check. If the backend needs to change this, it is recommended that the value be set higher than 5
minutes."
